# MERGE SORT PROJECT


## [16,21,11,8,12,22] Write the stages of the array according to the sort type : 

                    [16,21,11,8,12,22]
                    /                 \
               [16,21,11]          [8,12,22]
                  /   \               /    \
               [16] [21,11]       [8,12]   [22]
               /     |  \         /   |      /
              [16]  [21] [11]    [8] [12]  [22]
                     \                 /
                 [11,16,21]        [8,12,22]
                       \              /
                      [8,11,12,16,21,22]
                      
                      
## Write the Big-O notation : 

    O(nlogn) --> O(6log6)
